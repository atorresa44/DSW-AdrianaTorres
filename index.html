¿Qué es un OVI?

Se define Objeto Virtual Informativo (OVI) como un conjunto de recursos digitales que puede ser utilizado en diversos contextos educativos y que posee una estructura de información externa (metadato) para facilitar su almacenamiento, identificación y recuperación.

Papel del curso de programación web

El curso pretende preparar al alumno en la creación de OVI's, por medio de la preparación en el diseño de páginas web mediante el uso de HTML5 y CSS3, siendo esto fundamental en la era de las tecnologías y la información.

Construcción de un OVI

En la presente página, como muestra de un OVI, pretende presentarse contenido al público de manera instructiva; se elige dar información sobre GitHub, de manera integral, es decir, se presentará información mientras se ilusta la misma.

Objetivos del curso

En el desarrollo del curso de Diseños de sitios web se pretende dar las herramientas a los estudiantes para lograr el diseño de páginas web mediante el uso de lenguaje HTML y CSS.

Con el diseño de páginas web se pretende preparar al estudiante para afrontar la era de las TIC, ya que se permite el acceso al mayor medio de difusión de información de los últimos tiempos.


Git y GitHub

¿Qué es Git?

Es un software de control de versiones, diseñado pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando éstas tienen un gran número de archivos de código fuente. Git fue creado pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando éstas tienen un gran número de archivos de código fuente, es decir Git proporciona las herramientas para desarrollar un trabajo en equipo de manera inteligente y rápida y por trabajo se entiende algún software o página que implique código el cual se necesite realizarlo con un grupo de personas.

¿Què es GitHub?

Es un sitio web y un servicio en la nube que ayuda a los desarrolladores a almacenar y administrar su código, al igual que llevar un registro y control de cualquier cambio sobre este código. La interfaz de GitHub es bastante fácil de usar para el desarrollador novato que quiera aprovechar las ventajas del Git. Sin GitHub, usar un Git generalmente requiere de un poco más de conocimientos de tecnología y uso de una línea de comando.
