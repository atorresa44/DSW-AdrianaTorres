GIT y GITHUB

Bienvenidos al Objetivo Virtual de Información O.V.I. del curso de Diseño de Sitios Web, donde encontrará información introductoria sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web. La información acá presentada es muy importante porque le va a servir de guía para entender como funcionan las herramientas GIT y GITHUB y como se puede realizar la creación de un repositorio básico sincronizando ambos instrumentos.

Objetivo Principal

Dar a conocer a los estudiantes las herramientas Git y GitHub y como estas aportan a la obtención de conocimientos en el Diseño de Sitios Web.

Objetivos específicos

  • Instalar las herramientas Git y GitHub.
  • Crear un repositorio de prueba el cual debe estar sincronizado con el cliente de escritorio y su cuenta local de GITHUB
  • Diseñar un archivo con el nombre index.html y subirlo desde el computador al repositorio local de GITHUB para luego sincronizarlo con el remoto.

¿Qué es Git? Es un software de control de versiones, diseñado pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando éstas tienen un gran número de archivos de código fuente. Git fue creado pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando éstas tienen un gran número de archivos de código fuente, es decir Git proporciona las herramientas para desarrollar un trabajo en equipo de manera inteligente y rápida y por trabajo se entiende algún software o página que implique código el cual se necesite realizarlo con un grupo de personas. ¿Qué es GitHub? Es un sitio web y un servicio en la nube que ayuda a los desarrolladores a almacenar y administrar su código, al igual que llevar un registro y control de cualquier cambio sobre este código. La interfaz de GitHub es bastante fácil de usar para el desarrollador novato que quiera aprovechar las ventajas del Git. Sin GitHub, usar un Git generalmente requiere de un poco más de conocimientos de tecnología y uso de una línea de comando.
